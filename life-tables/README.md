# Life table data

These data come from the Human Mortality Database

https://www.mortality.org/Home/Index

in particular data for Sweden

https://www.mortality.org/Country/Country?cntr=SWE

I have retrieved for women

https://www.mortality.org/File/GetDocument/hmd.v6/SWE/STATS/fltper_1x5.txt

I wrote a script

[extract-life-tables.pl]|(extract-life-tables.pl)

that selected three five year intervals, 1800-1804, 1900-1904 and
2000-2005. There is a graph with survivorship curves in the essay.
